# Hotel Management System
A hybrid decentralized hotel booking management system built with React, Node.js, Express, and Algorand TestNet for tamper-proof reservation verification.

## Repository of Admin Panel
https://github.com/Nabia-Sheikh/hotel-management-admin

## Features
- Clean Design
- Responsive
- Featured Rooms
- Rooms list
- Room filteration
- Single page of Room
- Private Routes
- Authentication
- Validation
- Booking details page
- Booking tracking

Go to the project directory
```bash
  cd /workspaces/hotel-booking-dapp/server
  npm run dev
```
```bash
  cd /workspaces/hotel-booking-dapp/client
  npm run dev
```

Install dependencies
```bash
  npm install
```

Start the server
```bash
  npm start
```
